Admin Help

Quick Start

  1. Go to the Lobby tab and confirm Server URL, Room Code, Admin Passcode, and Rounds.
  2. Click Create/Update Room.
  3. Click Admin Login.
  4. Open Question Pool to choose which packs are in play.
  5. Open Game Settings to configure the match rules.
  6. Start the game from Question Pool, then watch the countdown in Game Monitor. If automatic issue is disabled, use Issue Question there.

Question Pack Setup

The easiest setup is to place JSON question packs directly into assets/questions/ on the server package. Each pack must use the .json extension. You can optionally define a single root "category" that applies to the whole pack.

Step 1: Create or copy one or more JSON pack files into assets/questions/.
Step 2: Restart the Quizter server so it reloads the available pack files.
Step 3: Open Question Pool Selection, expand a discovered category, and check the groups or individual packs you want in play.
Step 4: Confirm the status line shows the expected counts for available questions and questions in play.
Adjust all game rules in the Game Settings tab before creating or updating the room.
Round issue mode: leave Enable Automatic Timed Question Issue off to use the manual Issue Question button in Game Monitor, or turn it on and set the number of seconds between automatically issued questions.
If a pack has no root "category", Quizter will place it under Generic.

Question Pack Format

{ "category": "History", "questions": [ { "prompt": "Question text", "options": ["A", "B", "C", "D"], "correct_index": 1, "points": 100, "image_url": null } ] }

LLM Question Generation Prompt

Tip: an easy way to generate question packs is to ask an LLM such as ChatGPT with this prompt:

Please generate { } questions about { } using the JSON format provided in this example: { "category": "History", "questions": [ { "prompt": "Which document first established the principle that government derives its authority from the consent of the governed?", "options": [ "The Articles of Confederation", "The Declaration of Independence", "The Bill of Rights", "The Federalist Papers" ], "correct_index": 1, "points": 100, "image_url": null } ] }

Game Instructions

  • Players join from the QR code or join URL and answer timed questions in a browser.
  • The player view shows the category, timer, leaderboard, and round progress.
  • Correct and incorrect result graphics flash after each round, and incorrect flashes show the correct answer.
  • Power-ups can be enabled or disabled from the Game Settings tab.

Troubleshooting

  • If players cannot connect, confirm everyone is on the same local network and the firewall allows the server executable.
  • If the browser does not open automatically, manually visit http://127.0.0.1:8080/admin on the host machine.
  • If new question packs do not appear, restart the server after adding them to assets/questions/.
